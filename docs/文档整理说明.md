## 格式与链接

- 文档中心 [docs/README.md](README.md) 使用表格罗列各主题文档，并提供「按场景导航」。
- 子目录 `pyrosetta/`、`environments/`、`scripts/` 各有 README，统一采用「文档列表 + 相关文档」结构。
- 站内链接使用相对路径；指向 `docs/` 根目录文件时使用 `../`，指向项目根目录时使用 `../../`。
